# Nitesh Chavan — Cyber Security Portfolio v3

## What's included
- Terminal boot animation (NITESH_OS boot sequence)
- Custom green crosshair cursor with ring follower
- Hacker/Stealth mode toggle in the navbar
- Full SEO + Open Graph meta tags (Google, WhatsApp, LinkedIn previews)
- Matrix rain + scanline effects
- Typewriter hero with picoCTF stats
- Profile avatar card with TryHackMe photo + ONLINE status
- Animated stat counters
- Skill cards with progress bars
- CTF Battle Record (picoCTF — Noten: 43 challenges)
- Education & Certifications (TryHackMe, CEH, CompTIA)
- Security writeups section
- Contact form with backend API
- Active nav highlighting + scroll progress bar
- Back-to-top button
- Fully responsive

## Quick Start
cd frontend && npm install && npm run dev   # http://localhost:3000
cd backend  && npm install && npm run dev   # http://localhost:4000

## Deploy
- Netlify: drag frontend/dist/ after running npm run build
- Vercel: import GitHub repo, reads vercel.json automatically
- Railway/Render: deploy backend/ folder

## Update Your Info
- ContactSection.tsx: replace nitesh@example.com with your real email
- WriteupsSection.tsx: add real writeup URLs
- frontend/public/: add resume.pdf
- EducationSection.tsx: add new certs as you earn them
